# HTML Mastery

A structured daily learning repository focused on mastering HTML from fundamentals to advanced page structure through consistent practice, verification, and real projects.

---

## 🎯 Goal
Develop a strong first-principles understanding of HTML by building real, structured web pages through disciplined daily practice and continuous improvement.

---

## 📚 Learning Strategy
- Learn concepts step by step
- Apply immediately through hands-on exercises
- Solve challenges independently before checking solutions
- Upload only verified and working code
- Maintain one meaningful commit per day

---

## 📈 Progress Log

- Day 01 — HTML structure, headings, paragraphs, links, attributes, lists
- Day 02 — Images, file paths, inline vs block elements
- Day 03 — Semantic structure: header, main, section, footer
- Day 04 — Forms: labels, inputs, radio buttons, checkboxes, validation
- Day 05 — Tables: thead, tbody, tfoot, colspan
- Day 06 — Advanced forms: textarea, select, fieldset, legend
